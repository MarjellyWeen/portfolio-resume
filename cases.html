FestiFavs FrontEnd

This was a group project for school. The idea of this webapp was to be able to add your favorite festivals into a list that you can easily access through your account. We made this project twice. Once with Angular(Frontend) and once with Laravel(Backend).

  • General: Angular
  • Coding: Javascript/Typescript
  • Layout: HTML5
  • Styling: CSS3 With Bootstrap
  • Database: JSON server
  • Date: March 2020

  • General: Laravel
  • Coding: PHP(and additional Jquery)

Liliths Venture

  • General: Unity3D
  • Coding Language: C#
  • 3D models: 3ds Max
  • Graphics: Adobe Photoshop & Illustrator
  • Date: June 2019
