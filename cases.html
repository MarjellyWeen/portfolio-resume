FestiFavs FrontEnd

This was a group project for school. The idea of this webapp was to be able to add your favorite festivals into a list that you can easily access through your account. The advantage of Festifavs is that music lovers do not have to start a search on the internet, but can start from one place for all information. We made this project twice. Once with Angular(Front-end) and once with Laravel(Back-end).

  • General: Angular
  • Coding: Javascript/Typescript
  • Layout: HTML5
  • Styling: CSS3 With Bootstrap
  • Database: JSON server
  • Date: March 2020
  • Github Page: Front-end

Angular

As a general practice for coding with a javascript framework we decided on Angular.
That way could also get familiar with Typescript.
